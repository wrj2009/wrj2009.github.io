使用 Qt Designer 设计界面

在安装上面的库之后，在文件资源管理器中打开 Python 的安装目录。在我的电脑上，安装目录是 C:\Users\username\AppData\Local\Programs\Python\Python39。接着，打开这个目录下的 Lib\site-packages\PySide6。这个目录下有一个 designer.exe。打开它即可进入 Qt Designer。
打开后默认会弹出一个窗口。在左侧的“templates\forms”中选择 Widget，然后点击下方的“创建”按钮，就可以创建一个 UI 文件了。

创建之后，从左边随意拖动几个组件到中间的“Form - untitled”窗口中。组件的各种属性（样式）可以到右边的“属性编辑器”中编辑。
（几个常用的组件：普通按钮“Push Button”、单选框“Radio Button”、复选框“Check Box”、单行输入框“Line Edit”、显示文字的标签“Label”、进度条“Progress Bar”等。）
完成后，可以通过顶部菜单的“窗体▶预览于▶…”或者“窗体▶预览”来预览设计的用户界面。满意后，点击顶部菜单“文件▶另存为…”来保存 UI 文件。
